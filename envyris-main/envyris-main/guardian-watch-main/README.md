## 📖 What is Envyris?
**Envyris** (built around the `guardian-watch-main` module) is an intelligent platform designed to track, monitor, and detect anomalies or critical risks in real-time. It acts as a digital sentinel for processing incoming system alerts, environmental data, or security feeds.

## 🛠️ How It Works
1. **Data Ingestion:** The core engine continuously monitors active data streams.
2. **Threat Assessment:** It evaluates incoming data points against preset thresholds to flag vulnerabilities or abnormal patterns.
3. **Instant Notification:** Once an anomaly is detected, the built-in alert-sender system dispatches immediate warnings to system administrators.

## 💡 Why It Is Used
* **Proactive Security:** It catches issues the exact second they happen instead of generating delayed logs hours later.
* **Automated Dispatches:** Minimizes human response delay by automating warning dispatches.
* **High Performance Engine:** Optimized using lightweight development tools to process high-throughput data pipelines seamlessly.

## 🧰 Tech Tools Used
Based on the system architecture, Envyris utilizes a fast, modern runtime and development stack:
* **Bun:** Employed as a rapid engine for managing project locks (`bun.lockb`) and running dependencies.
* **Vite & React:** Powers the highly reactive, fast-rendering user interface components.
* **TypeScript:** Provides structured, clean, and bug-free code compilation.
* **Tailwind CSS:** Manages the modern, fluid, responsive visual styles.
